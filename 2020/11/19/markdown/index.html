|阅读量:|评论数:

Markdown 和 html

Markdown是html标签的子集，Markdown不会在块级html标记中进行处理

自动转义特殊字符

例如：<、&

段落和换行符

1、一段是一个或者多个连续的文本行，由一个或多个空行分隔
2、仅包含空格或者制表符的行被视为空白
3、普通段落不应该以空格或者制表符缩进

Markdown 语法

标题

# h1
## h2
…

加粗

**text**

我是粗体

引用

>

有序列表

1. 第一个
2. 第二个
3. 第三个

无序列表

- 第一个
- 第二个
- 第三个

代码

`这里填代码`

代码块

```这是代码块```

连接

[title](https://www.example.com)

https://www.baidu.com

图片

![alt text](image.jpg)
