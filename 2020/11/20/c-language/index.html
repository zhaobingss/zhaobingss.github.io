C语言学习笔记

标准ANSI

C89、C99
C89 要求将变量的定义放到语句的前面，这个非常恶心

特点

  • 代码级跨平台，不同操作系统需要重新编译
  • 允许访问物理地址，对硬件进行操作
  • 过程式的语言
  • 传值和指针
  • 支持预处理

C语言运行机制

  • 编辑
  • 编译
  • 连接
  • 运行

编译、连接(VS)

  • cl.exe
  • link.exe

C转义字符

  1. \t 制表符
  2. \n 换行符
  3. \\ 一个\
  4. \’ 一个’
  5. \" 一个双引号
  6. \r 一个回车

注释

  1. // 行注释
  2. /** 块注释 **/

变量

  • 变量是程序的基本组成单位

  • 变量详单与内存中一个数据存储空间的表示，通过变量名可以访问到变量值

    int a = 0;
    double b = 1.3;
    char* str = “hello world”

数据类型

  • 每一种数据都定义了明确的数据类型，在内存中分配不同大小的内存空间（使用字节(byte)多少表示， 使用sizeof来计算）
  • 数据类型的表示范围遵循左闭右开([))的原则
  • 基本类型
    • 字符类型 char %c或%d 1字节
    • 数值类型
      • 整型
        • short %d 2字节
        • int %d 2或者4字节
        • long %ld 4字节(win和linux不一样)，严格来说这并不是一种数据类型，只是long int的缩写
        • long long %lld 8个字节，严格来说这并不是一种数据类型，只是long long int的缩写
      • 浮点型
        • float %.6f 4字节 6位小数 声明加f才代表单精度小数
        • double %.15f 8字节 15位小数 (默认)
        • 浮点数 = 符号位 + 指数位 + 尾数位 例如：1.2E38，浮点数都是近似值
        • 尾数部分可能丢失，造成精度损失
  • 构造类型
    • 数组 [10]
    • 结构体 struct
    • 共用体 union
    • 枚举 enum
  • 布尔 0假-非0真
  • 指针类型 *
  • 空类型 void
  1. 在C中没有字符串使用*字符数组*表示字符串
  2. 在不同的操作系统、位数(主要是32位和64位的区别)和编译器上部分数据类型的字节大小是不一样的
  3. char、short、int、long 分有符号(signed默认)和无符号(unsigned)

数据类型转换

自动类型转换

  • 在进行赋值或者运算时，精度小的自动转换为精度大的
    • 排序 char ->int->unsigned int->long->unsigned long->float->double->long double
    1. 有多种类型的数据进行混合运算时，系统首先将所有数据转换成最大的数据类型，然后进行运算
    2. 若两种类型的字节数不同，转化成字节数大的类型，若字节数相同，且一种有符号，一种无符号，则转换成无符号
    3. 赋值时右边转左边，数据可能会丢失精度，丢失部分四舍五入

强制类型转换

  • (类型)表达式 和go有差别
    • 强制类型转换不改变操作数本身
    • 只针对最近的操作数有效
